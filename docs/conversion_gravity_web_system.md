3. **Automated Nurture**: Immediate handoff to automated email/SMS sequences after lead capture.

## 3. Workflow for Haul Command Assets

*   **Step 1: Benchmark**: Find 3 premium inspirations on Dribbble/Mobbin.
*   **Step 2: Copy**: Generate high-intent copy using Google AI Studio (Gemini 1.5 Pro).
*   **Step 3: Build**: Use Cursor to build the front-end (Vite/Next.js) or GHL Page Builder for standard funnels.
*   **Step 4: Automate**: Link all forms to the **Shared Spine** (Supabase) + GHL workflows.
*   **Step 5: Voice**: Generate any required onboarding/explainer audio via ElevenLabs.
