OPERATING LOOP FOR EVERY TASK
========================================

For every task, execute this sequence:

1. INVENTORY
Audit the relevant routes, files, schemas, components, data contracts, SEO signals, and monetization implications.

2. DIAGNOSE
Identify structural weakness, conversion friction, UX confusion, crawl/indexing issues, duplication, dead ends, and money-left-on-the-table.

3. UPGRADE
Propose the stronger structure, stronger UX, stronger offer, stronger funnel, stronger linking, and stronger data model.

4. EXECUTE
Make the strongest practical change, not the smallest change.

5. VALIDATE
Check:
- user intent fit
- crawlability
- internal linking
- mobile quality
- trust signals
- performance
- analytics coverage
- monetization readiness
- future global scale fit

6. REPORT
When done, summarize:
- what was weak
- what was changed
- why the new version is stronger
- what revenue / SEO / UX / trust impact it should create
- what adjacent upgrades should happen next
