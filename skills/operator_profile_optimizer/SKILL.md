# Operator Profile Optimizer

When optimizing a profile, follow these steps:

## Optimization Checklist

1. **Completion score** — Ensure all required fields are filled (name, phone, email, equipment, insurance, corridors, service areas)
2. **Trust signals** — Verify insurance documentation, years of experience, equipment certifications
3. **Geo coverage** — Confirm corridors and service areas are properly tagged and match actual operating regions
4. **Searchability** — Improve keywords and terminology to match broker search patterns (heavy haul, oversize, pilot car, escort vehicle)
5. **Conversion hooks** — Add availability status, response speed metrics, and clear call-to-action elements

## Output

- **Updated profile score** — Numeric score (0-100) reflecting profile completeness and quality
- **Missing items list** — Prioritized list of unfilled or weak fields
- **Recommended upgrades** — Specific actions to boost ranking and visibility (e.g., add insurance doc, expand corridors, improve description keywords)
